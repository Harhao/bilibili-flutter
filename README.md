### 三、安装依赖并运行app
  - 安装依赖:尽量设置安装为国内镜像，境外安装比较慢，而且容易出现安装失败情况
    ```bash
    flutter pub get
    ```
  - 运行app
    ```
    	flutter run 
    ```
